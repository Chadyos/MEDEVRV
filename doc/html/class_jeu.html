void Jeu::tourDeJeu ( )

Méthode qui gère le déroulement d'un tour de jeu.

Cette méthode permet de gérer les actions effectuées lors d'un tour de jeu. Elle demande le nombre de joueurs et le nombre de bateaux, initialise les joueurs, puis effectue les actions de chaque joueur tour à tour jusqu'à ce qu'un joueur gagne.
